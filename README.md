# 🌱 EcoVision AI

EcoVision AI is a Gradio-based AI application that performs environmental text analysis using Hugging Face Transformers.

---

## 🚀 Features

- Environmental sentiment analysis
- AI-based environmental poster generation
- Text summarization
- Fill-mask word prediction
- Interactive Gradio interface

---

## 🛠 Technologies Used

- Python
- Gradio
- Hugging Face Transformers
- PyTorch
- Pillow
